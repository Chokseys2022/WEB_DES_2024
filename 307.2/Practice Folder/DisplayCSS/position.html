CSS Position Property Examples

Position: static

The `static` value is the default positioning. The element is positioned according to the normal document flow.

I am static

Position: absolute (with relative parent)

The `absolute` value positions the element relative to its nearest positioned ancestor (a parent with `position: relative` in this case). This prevents it from overlapping unrelated elements.

I am absolute

Position: absolute

The `absolute` value positions the element relative to its nearest positioned ancestor (other than `static`) or the document if no ancestor is positioned.

I am absolute

Position: fixed

The `fixed` value positions the element relative to the viewport, so it stays in the same place even when the page is scrolled.

I am fixed

Position: sticky

The `sticky` value toggles between `relative` and `fixed`, depending on the scroll position. It sticks to a defined position until the containing block is scrolled out of view.

I am sticky
