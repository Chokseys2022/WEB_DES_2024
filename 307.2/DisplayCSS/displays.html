CSS Display Property Examples

Display: block

The `block` display value makes an element take up the full width and start on a new line. Examples include `

Display: inline

The `inline` display value keeps the element inline, only as wide as its content. It does not force a new line.

I am inline

Display: run-in

The `run-in` display value is context-sensitive, acting as either inline or block depending on the parent element's context.

I am run-in

Display: flow

The `flow` value is the default layout behavior for block-level elements, following the normal document flow.

I am flow

Display: flow-root

The `flow-root` value establishes a new block formatting context, useful for clearing floats.

I am flow-root

The `table` value makes the element behave like a table container, which can include rows and cells.

Display: flex

The `flex` value enables the Flexbox layout, allowing child elements to align along a flexible axis with customizable spacing.

Display: grid

The `grid` value activates the CSS Grid layout, creating a structured layout with rows and columns.

The `ruby` value is used for ruby annotations, commonly seen in East Asian texts. It allows combining base characters and annotations.

Display: block flow

The `block flow` combination sets the element as a block-level container using the normal flow layout. This is the default for most block elements.

Block Flow

Display: inline table

The `inline table` value allows the element to behave like a table but remain inline with other elements.

Inline Table

Display: flex run-in

The `flex run-in` value is experimental and combines Flexbox layout with context-sensitive run-in behavior.

Flex Run-In

Display: contents

The `contents` value removes the element’s box but keeps its child elements in the layout, as if the parent doesn’t exist.

Contents Example

Display: none

The `none` value hides the element completely, removing it from the layout and the document flow.

This will not be visible

Display: inherit

The `inherit` value makes the element take its display value from its parent element.

Inherit Example

Display: initial

The `initial` value resets the element’s display property to its default, as defined by the browser.

Initial Example

Display: unset

The `unset` value acts as `inherit` if the parent has a defined value or `initial` if it doesn’t.

Unset Example
